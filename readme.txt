PayFlexi Installment Payment Gateway for Easy Digital Downloads allows you to accept installment or full payments from local and international customers before getting access to downloads

== Description ==
<strong>You have to set the Callback URL and the Webhook URL in the [API Keys & Webhooks](https://merchant.payflexi.co/settings/#api-keys-integrations) settings page in your Paystack account</strong>. Go to the plugin settings page for more information.
This is a PayFlexi Flexible Checkout payment gateway for Easy Digital Downloads.
